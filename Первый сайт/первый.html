Работа с таблицами.

Главным тегом является "table", внутри которого должны лежать теги "td"- которые создадут ячейки.

Также тег "table", должен иметь атрибут "border", который задает границу таблице и ее ячейкам.

Таблица строится строками сверху вниз. Поэтому, если простым языком, то tr- это строчка, td- это столбик. А th это, тот же самый td стобик, но выделенный жирным, как оглавление столбца.

Пример таблицы с тремя рядами и тремя ячейками в каждом ряду:

ячейка 1 ячейка 2 ячейка 3
ячейка 4 ячейка 5 ячейка 6
ячейка 7 ячейка 8 ячейка 9

Мы сделали ее с помощью следующего кода:

<table border="1"> <!--Это будет первый
ряд таблицы:-->  <tr> <td>ячейка 1</td> <td>ячейка 2</td> <td>ячейка 3</td> </tr> <!--Это будет второй
ряд таблицы:-->  <tr> <td>ячейка 4</td> <td>ячейка 5</td> <td>ячейка 6</td> </tr> <!--Это будет третий
ряд таблицы:-->  <tr> <td>ячейка 7</td> <td>ячейка 8</td> <td>ячейка 9</td> </tr> </table>

А теперь, увеличим таблицу, зададав как и картинке ширину и высоту. Вставив после border="1": width и height.

Начинаем работать со стилями css.

Каждому тегу HTML, соответствует селектор CSS. Если в html, будет тег p (в скобках), то ему будет соответствовать селектор p.

p { color: red; }

И так с каждым элементом, будь то h1, h2 или h3.

Также можно менять цвет через RGB и #. Генераторы цвета: ColorSheme, ColorCop

Ширина и высота элементов на CSS.

Для задания ширины и высоты элементов в CSS предназначены свойства width и height. Задаются они в пикселях- px.

Выравнивание текста на CSS.

Для этого, используется свойство text-align, дабывляю к нему значения left, right, center и justify (одноременно по правому и левому краю).

Значение left для абзацев можно не ставить, т.к. они по умолчанию слева. Но элементы с умолчанием по центру, такие как th, можно выравнять по левому краю.

Жирность текста на CSS.

За жирность всего текста отвечает свойство - font-weight, и значение bold (делает текст жирным), и значение normal (делает текст нормальным).

Курсив на CSS.

За курсив всего текста отвечает свойство - font-style, и значения italic (делает текст курсивом), и значение normal (делает текст нормальным).

Размер шрифта на CSS.

За размер всего текста отвечает свойство - font-size, и меняется в пикселях (px).

Тип шрифта на CSS.

За шрифт всего текста отвечает свойство - font-family, и вот основные шрифты: Times New Roman (стоит по умолчанию), Arial, Georgia, Verdana, Trebuchet MS, Impact, Arial Black, Comic Sans MS, Courier New.

Межстрочный интервал на CSS.

За межстрочный интервал текста (расстояние между линиями текста), отвечает свойство line-height, задающее высоту линии текста.

Но, реальный видимый промежуток будет вычисляться так: line-height - font-size =видимое расстояние между строками текста.

Значение свойства line-height можно записать как число или дробь. К примеру, font-size равен 10px, а line-height - 1.5. В этом случае реальное значение line-height будет 10px * 1.5 = 15px. Ну, а видимый белый промежуток между линиями текста будет 5px: 15px - 10px = 5px (Преимущество такого способа задания line-height в том, что при изменении размера шрифта автоматически будет меняться и межстрочный интервал.).

Свойство-сокращение font.

В CSS для большего удоства существует специальное свойство-сокращение font. Данное свойство позволяет одновременно задать размер шрифта, семейство, жирность, курсив и межстрочный интервал.

bold italic 16px/50px Arial

Доп. подмечание: обязательно должно быть семейство шрифта.

Красная строка текста на CSS.

За отступ первой строки текста, т.е. за красную строку отвечает свойство - text-indent.

Эффекты текста на CSS.

Свойство text-decoration - позволяет задавать некоторые эффекты для текста: underline (подчеркивание), line-through (перечеркивание), overline (линию сверху), и отменять эти эффекты (none).

Например, в тексте убираем подчеркивание с ссылки на Гугл.

Тег span для кусочков текста.

Также и для любого кусочка текста или элемента. Просто открываем SPAN и закрываем /span> (так я и покрасил этот кусок и изменил его размер).

Тег div для группировки тегов.

Помимо тега class, есть специальный тег div class="НАЗВАНИЕ"> (и закрываем через /div>), который позволяет сгруппировать несколько тегов, чтобы затем сними сделать что-нибудь одновременно. Т.е. по сути мы тоже выделяем кусок, но куда больше (с несколькими тегами p, к примеру).

Группировка классов.

Группировка классов в CSS, происходит за счет их перечисления:

  1. .НАЗВАНИЕ, .НАЗВАНИЕ2 {
  2. color: red;
  3. }

Тег с заданным классом.

Суть в том, что можно, обратиться к конкретному тегу с классом. К примеру, можно выбрать все h2 с классом eee и отдельно выбрать все p с этим классом. Для этого в селекторе нужно написать имя тега, а затем слитно имя класса.

  1. h2.НАЗВАНИЕ {
  2. color: red;
  3. >}
  4. p.НАЗВАНИЕ {
  5. >color: green;
  6. }

Распространенная ошибка новичков.

Разница в работе с селекторами следующая: p.eee - такой селектор выбирает абзацы с классом eee. А вот если я сделаю так: p .eee, то выберу все элементы с классом eee, находящиеся внутри абзацев.

Элемент с несколькими классами.

Можно одновременно задать несколько CSS классов какому-нибудь тегу. Для этого в атрибуте class эти классы нужно перечислить через пробел. Выглядеть это будет так:

  1. p class="chel pups">
  2. lorem ipsum
  3. /p>

Выбор элемента по уникальному id на CSS.

Кроме классов, можно также выбирать элементы по id (например этому кусочку), представляющим собой уникальный идентификатор элемента. Уникальный идентификатор задается с помощью атрибута id, в котором пишется придуманное нами имя. К примеру:

  1. div id="block1">
  2. p>текст/p>
  3. /div>

Чтобы обратитьсяк элементу с заданным id:

  1. #block1 {
  2. color: red;
  3. }

    1. div id="lok2" class="chel">
